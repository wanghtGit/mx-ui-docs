MxSteps 步骤条 ​

引导用户按照流程完成任务的分步导航条， 可根据实际应用场景设定步骤，步骤不得少于 2 步。

基础用法 ​

MxSteps可以通过 onUp 、 onDown 属性绑定的方法的返回值，控制上下步的切换

onUp 、 onDown 绑定的方法，需要有返回值，可以返回 Boolean 类型的值，当返回 true 时，能够上一步、下一步切换，反之无法切换步骤

onUp 、 onDown 绑定的方法，也可以返回一个 Promise 类型的值，当步骤切换，涉及到一些异步操作时，需要返回 Promise 类型值来控制。

Show Code
vue
<template>
  <div class="step-wrapper">
    <mx-steps
      finish-status="success"
      :onUp="doUpClick"
      :onDown="doDownClick"
      :onFinish="doFinish"
    >
      <mx-step title="用户需知" >
        <span>用户需知......</span>
      </mx-step>
      <mx-step title="信息采集">
        <span>信息采集......</span>
      </mx-step>
      <mx-step title="查看执行">
        <span>查看执行......</span>
      </mx-step>
      <mx-step title="完成" state="success">
        <div style="text-align: center; margin-top: 100px">
          <mx-icon type="CircleCheckFilled" :size=80 color="rgba(86, 196, 26, 0.8)"></mx-icon>
          <h2>操作成功</h2>
          <span style="color: gainsboro">你已经成功完成信息填写，请等待反馈</span>
        </div>
      </mx-step>
    </mx-steps>
  </div>
</template>
<script setup>
// import { MxNotify } from '@wxct/mx-ui'
import * as MxUI from '@wxct/mx-ui'

const { MxNotify } = MxUI

const doUpClick = (stepIndex) => {
  // stepIndex 为当前步骤的索引index

  console.log('doUpClick')

  if (stepIndex === 1) {
    return new Promise((resolve) => {
      setTimeout(() => {
        MxNotify.success('已返回上一步！')
        resolve()
      }, 2000);
    })
  } else {
    return true
  }
}
const doDownClick = () => {
  console.log('doDownClick')

  return true
}
const doFinish = () => {
  console.log('doFinish')

  MxNotify.success('步骤完成！！！')
}
</script>
<style scoped>
.step-wrapper {
  width: 600px;
  height: 500px;
}
</style>
