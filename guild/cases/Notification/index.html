MxNotify 通知 ​

悬浮出现在页面角落，显示全局的通知提醒消息。

基础用法 ​

MxUI 注册了 $MxNotify 方法并且它接受一个 Object 作为其参数。 在最简单的情况下，你可以通过设置 title 和 message 属性来设置通知的标题和正文内容。 默认情况下，通知在4500毫秒后自动关闭，但你可以通过设置 duration 属性来自定义通知的展示时间。 如果你将它设置为 0，那么通知将不会自动关闭。 需要注意的是 duration 接收一个 Number，单位为毫秒。

Show Code
vue
<template>
  <mx-button plain @click="open1"> Closes automatically </mx-button>
  <mx-button plain @click="open2"> Won't close automatically </mx-button>
</template>

<script lang="ts" setup>
import { h } from 'vue'
// import { MxNotify } from '@wxct/mx-ui'
import * as MxUI from '@wxct/mx-ui'

const { MxNotify } = MxUI

const open1 = () => {
  MxNotify({
    title: 'Title',
    message: h('i', { style: 'color: teal' }, 'This is a reminder'),
  })
}

const open2 = () => {
  MxNotify({
    title: 'Prompt',
    message: 'This is a message that does not automatically close',
    duration: 0,
  })
}
</script>

不同类型的通知 ​

我们提供了四种不同类型的提醒框：success、warning、info 和error。

MxUI 为 Notification 组件准备了四种通知类型：success, warning, info, error。 他们可以设置 type 字段来修改，除上述的四个值之外的值会被忽略。 同时，我们也为 Notification 的各种 type 注册了单独的方法，可以在不传入 type 字段的情况下像 open3 和 open4 那样直接调用。

Show Code
vue
<template>
  <mx-button plain @click="open1"> Success </mx-button>
  <mx-button plain @click="open2"> Warning </mx-button>
  <mx-button plain @click="open3"> Info </mx-button>
  <mx-button plain @click="open4"> Error </mx-button>
</template>

<script lang="ts" setup>
// import { MxNotify } from '@wxct/mx-ui'
import * as MxUI from '@wxct/mx-ui'

const { MxNotify } = MxUI

const open1 = () => {
  MxNotify({
    title: 'Success',
    message: 'This is a success message',
    type: 'success',
  })
}

const open2 = () => {
  MxNotify({
    title: 'Warning',
    message: 'This is a warning message',
    type: 'warning',
  })
}

const open3 = () => {
  MxNotify({
    title: 'Info',
    message: 'This is an info message',
    type: 'info',
  })
}

const open4 = () => {
  MxNotify({
    title: 'Error',
    message: 'This is an error message',
    type: 'error',
  })
}
</script>

自定义消息弹出的位置 ​

可以让 Notification 从屏幕四角中的任意一角弹出

使用 position 属性设置 Notification 的弹出位置， 支持四个选项：top-right、top-left、bottom-right 和 bottom-left， 默认为 top-right。

Show Code
vue
<template>
  <mx-button plain @click="open1"> Top Right </mx-button>
  <mx-button plain @click="open2"> Bottom Right </mx-button>
  <mx-button plain @click="open3"> Bottom Left </mx-button>
  <mx-button plain @click="open4"> Top Left </mx-button>
</template>

<script lang="ts" setup>
// import { MxNotify } from '@wxct/mx-ui'
import * as MxUI from '@wxct/mx-ui'

const { MxNotify } = MxUI

const open1 = () => {
  MxNotify({
    title: 'Custom Position',
    message: "I'm at the top right corner",
  })
}

const open2 = () => {
  MxNotify({
    title: 'Custom Position',
    message: "I'm at the bottom right corner",
    position: 'bottom-right',
  })
}

const open3 = () => {
  MxNotify({
    title: 'Custom Position',
    message: "I'm at the bottom left corner",
    position: 'bottom-left',
  })
}

const open4 = () => {
  MxNotify({
    title: 'Custom Position',
    message: "I'm at the top left corner",
    position: 'top-left',
  })
}
</script>

有位置偏移的通知栏 ​

能够设置偏移量来使 Notification 偏移默认位置。

Notification 提供设置偏移量的功能，通过设置 offset 字段，可以使弹出的消息距屏幕边缘偏移一段距离。 注意在同一时刻，每一个的 Notification 实例应当具有一个相同的偏移量。

Show Code
vue
<template>
  <mx-button plain @click="open"> Notification with offset </mx-button>
</template>

<script lang="ts" setup>
// import { MxNotify } from '@wxct/mx-ui'
import * as MxUI from '@wxct/mx-ui'

const { MxNotify } = MxUI

const open = () => {
  MxNotify.success({
    title: 'Success',
    message: 'This is a success message',
    offset: 100,
  })
}
</script>

使用 HTML 片段作为正文内容 ​

message 支持传入 HTML 字符串来作为正文内容。

将 dangerouslyUseHTMLString 属性设置为 true，message 属性就会被当作 HTML 片段处理。

Show Code
vue
<template>
  <mx-button plain @click="open"> Use HTML String </mx-button>
</template>

<script lang="ts" setup>
// import { MxNotify } from '@wxct/mx-ui'
import * as MxUI from '@wxct/mx-ui'

const { MxNotify } = MxUI

const open = () => {
  MxNotify({
    title: 'HTML String',
    dangerouslyUseHTMLString: true,
    message: '<strong>This is <i>HTML</i> string</strong>',
  })
}
</script>

WARNING

message 属性虽然支持传入 HTML 片段，但是在网站上动态渲染任意 HTML 是非常危险的，因为容易导致 XSS 攻击。 因此在 dangerouslyUseHTMLString 打开的情况下，请确保 message 的内容是可信的，永远不要将用户提交的内容赋值给 message 属性。

全局方法 ​

MxUI 为 app.config.globalProperties 添加了全局方法 $MxNotify 。 因此在 Vue instance 中可以采用本页面中的方式调用 Notification。

单独引用 ​

js
import { MxNotify } from '@wxct/mx-ui'
1

你可以在对应的处理函数内调用 MxNotify(options) 来呼出通知栏。 我们也提前定义了多个 type 的单独调用方法，如 MxNotify.success(options)。 当你需要关闭页面上所有的通知栏的时候，可以调用 MxNotify.closeAll() 来关闭所有的实例。
