## 属性
参数 | 说明 | 类型 | 默认值
-- | -- | -- | --
allow | 能否编辑 | R：不能编辑，为禁用状态<br>W：可以编辑<br>X：不可编辑，且不展示文本框组件样式，只展示对应的值 | X
row | 文本域输入框行数 | number | 2
maxLength | 同原生 maxlength 属性 | string / number | '200'
